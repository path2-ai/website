# refinery x bricks

bricks and refinery are natively integrated, such that you can search for interesting bricks directly in your refinery application. This works for any programmable interface within refinery.

For instance, let's say you want to calculate the sentence complexity of a record, and you found the following brick:

![](https://kern-assets.s3.amazonaws.com/website/screenshots/brick-sentence-complexity.png)

In refinery, you can simply head over to the attribute calculation tab, and search for the "sentence complexity" brick.

![](https://kern-assets.s3.amazonaws.com/website/screenshots/refinery-x-bricks.png)

A wizard will help you configure the brick such that you can fetch the required code with ease.

![](https://kern-assets.s3.amazonaws.com/website/screenshots/refinery-sentence-complexity.png)
