# Roadmap

We're aiming to grow bricks continuously, adding circa 2 - 3 bricks per sprint (which is two weeks). If you have an idea for a brick, or generally would like us to implement one, feel free to open an [issue](https://github.com/code-kern-ai/bricks/issues).
