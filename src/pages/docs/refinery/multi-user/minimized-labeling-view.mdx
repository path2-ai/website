# Minimized labeling view

For experts and annotators, there is a specified, minimized labeling view:

![](https://files.readme.io/b6a7047-expert-or-annotator-labeling-view.png)

They can only work on the labeling tab, and they can select from available data slices that were created for them. They can still add comments though, which can help you as the engineer to better understand their thoughts in an asynchronous labeling session.
