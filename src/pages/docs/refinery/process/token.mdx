# Continuous API token (not in open-source version)

If you want to set up a continuous stream to an API, you can define project-specific personal access token in refinery. Simply head over to the tab, and click on `Add token`.
![](https://kern-assets.s3.amazonaws.com/website/screenshots/refinery-token.png)

You can configure the characteristics for that token. Make sure to backup / save the secret token on some trusted and sure location.
